## 🚀 API 服务器

### 启动服务：
```bash
python main.py
```

### 访问地址：
- API 文档：http://localhost:8000/docs
- 健康检查：http://localhost:8000/health
- 最新 CVE：http://localhost:8000/api/v1/cves/latest
- 统计摘要：http://localhost:8000/api/v1/stats/summary

## 📌 注意事项

1. **依赖安装**：
   - 首次运行会自动安装 `aiohttp` 库
   - 如需手动安装：`pip install aiohttp`

2. **API 限制**：
   - NVD API 有请求频率限制
   - 无 API Key：10 请求/分钟
   - 有 API Key：100 请求/分钟
   - 申请免费 API Key：https://nvd.nist.gov/developers/request-an-api-key

3. **数据存储**：
   - 所有采集的数据保存在 `cve_data/` 目录
   - 文件格式：`cves_YYYYMMDD_HHMMSS.json`

4. **性能建议**：
   - 建议一次采集不超过 30 天的数据
   - 大量数据采集请使用 API Key

## 🔧 故障排除

### 问题 1：无法连接到 NVD
- 检查网络连接
- 确认能访问 https://nvd.nist.gov
- 考虑使用代理

### 问题 2：GUI 窗口无法显示
- 确保使用 Python 3.10+ 版本
- Windows 系统需要 tkinter 支持（默认包含）

### 问题 3：Web 界面无法访问
- 检查端口 8080 是否被占用
- 尝试其他端口：`python -m http.server 8081`

## ⚙️ 配置检查

### Redis连接状态

**检查方法**:
- 查看GUI窗口顶部状态栏
- 如果显示 "⚠ Redis未连接"，程序会使用SQLite模式（正常）

**启动Redis**（可选）:
```bash
# 使用Docker启动
docker-compose up -d redis

# 检查状态
docker exec cve-redis redis-cli -a defaultpassword PING
```

### NVD API Key状态

**检查方法**:
- 查看GUI顶部状态栏
- 如果显示 "⚠ 未配置API Key"，采集速度会较慢

**配置API Key**（推荐）:
```bash
# 编辑.env文件
echo "NVD_API_KEY=your_api_key_here" >> .env

# 重启程序生效
```

**速度对比**:
- 无API Key: ~6秒/请求
- 有API Key: ~0.6秒/请求 (10倍提升)

**获取API Key**: https://nvd.nist.gov/developers/request-an-api-key

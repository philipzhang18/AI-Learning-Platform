### 1. **更强大的数据管理**
- **SQLite 数据库**: 高效的本地存储和查询
- **增量更新**: 避免重复采集相同的 CVE

- **实时搜索**: 支持多字段搜索和过滤
- **离线模式**: 完全支持无网络环境使用
- **数据导入**: 支持 CSV 格式数据导入

### 4. **更高的性能**
- **API Key 支持**: 采集速度提升 10 倍
- **异步处理**: 不阻塞界面操作
- **智能分批**: 避免 API 限制和超时
